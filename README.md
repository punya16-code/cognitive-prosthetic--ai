# Cognitive Prosthetic AI

## 🧠 Problem
Dementia patients struggle with memory loss, missed medications, and loneliness. Existing assistants are reactive and not personalized.

## 💡 Solution
We are building an AI-powered cognitive prosthetic that provides:
- Proactive reminders based on context
- Personalized memory recall using caregiver inputs
- Emotion-aware conversations
- Caregiver alert system

## ⚙️ Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- AI: LLM + Speech-to-Text + Text-to-Speech
- IoT: ESP32 (simulated for demo)

## 🚀 Features
- Context-aware voice nudges
- Infinite patience conversational AI
- Memory archive (RAG-based)
- Mood monitoring and alerts

## 🎯 Goal
To create an empathy-first AI companion that supports independence, safety, and emotional well-being for dementia patients.
